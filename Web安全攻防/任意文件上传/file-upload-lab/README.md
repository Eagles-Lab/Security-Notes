# Webshell 示例文件

本目录包含用于演示文件上传漏洞的Webshell示例文件。

## 文件列表

### 1. shell.php
**完整功能的Webshell**
- 显示服务器信息
- 执行系统命令
- 用于Level 1演示

**使用方法：**
1. 上传到服务器
2. 直接访问该文件
3. 在输入框中输入系统命令

### 2. shell_simple.php
**一句话木马**
- 最简洁的Webshell形式
- 需要配合客户端工具使用

**使用方法：**
- 使用中国菜刀、蚁剑、哥斯拉等工具连接
- URL: `http://localhost/file-upload-lab/shells/shell_simple.php`
- 密码: `cmd`

### 3. shell.gif
**GIF图片马**
- 包含GIF文件头
- 用于绕过文件头检测

**使用方法：**
1. 上传到Level 5
2. 通过文件包含漏洞访问
3. 访问: `shells/include.php?file=../uploads/shell.gif`

### 4. include.php
**文件包含漏洞演示页面**
- 用于演示图片马的利用
- 存在文件包含漏洞

**使用方法：**
1. 上传图片马
2. 访问 `shells/include.php?file=../uploads/图片马.gif`
3. 观察PHP代码被执行

## 安全警告

⚠️ **这些文件仅用于安全学习和漏洞演示！**

- 请勿在生产环境使用
- 请勿用于非法用途

## 学习目标

通过这些示例文件，学生可以：
1. 理解文件上传漏洞的危害
2. 学习Webshell的工作原理
3. 掌握图片马的制作和利用
4. 理解文件包含漏洞的利用方式
