# 测试文件说明

这个目录包含用于测试文件包含漏洞的示例文件。

## 文件列表

### test.txt
- 普通文本文件
- 用于测试基本的文件包含功能
- 攻击示例：`?file=files/test.txt`

### config.php
- 模拟的配置文件，包含敏感信息
- 包含数据库密码、API密钥等
- 直接包含会执行PHP代码，看不到源码
- 需要使用 php://filter 协议读取源码

**读取源码的方法：**
```
?file=php://filter/read=convert.base64-encode/resource=files/config.php
```

然后对返回的base64字符串进行解码，就能看到配置文件中的敏感信息。

## 安全提示

在真实环境中：
1. 永远不要在配置文件中硬编码敏感信息
2. 使用环境变量存储密钥
3. 配置文件应放在Web根目录之外
4. 对用户输入进行严格过滤和验证
